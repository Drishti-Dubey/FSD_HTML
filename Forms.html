Creating and Using Forms in HTML

Forms are essential for collecting user input in HTML. They are created using the <form> element. Inside a form, various types of input fields like text boxes, radio buttons, checkboxes, and more are used. Let's explore the creation and usage of forms with examples.

1. Basic Form

A basic form consists of an opening <form> tag and a closing </form> tag. Inside, different types of form elements such as text fields, radio buttons, checkboxes, and submit buttons can be added.

Example 1: A simple form for user login.

<form action="/submit-login" method="POST">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username" required><br>

    <label for="password">Password:</label>
    <input type="password" id="password" name="password" required><br>

    <button type="submit">Login</button>
</form>

2. Form with Multiple Input Types

Forms can include various types of input fields such as text, radio buttons, checkboxes, dropdown menus, and more.

Example 2: A form to collect user feedback, including text input, radio buttons, and a dropdown menu.

<form action="/submit-feedback" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" required><br>

    <label>Gender:</label>
    <input type="radio" id="male" name="gender" value="male"> Male
    <input type="radio" id="female" name="gender" value="female"> Female<br>

    <label for="rating">Rating:</label>
    <select id="rating" name="rating">
        <option value="1">1</option>
        <option value="2">2</option>
        <option value="3">3</option>
        <option value="4">4</option>
        <option value="5">5</option>
    </select><br>

    <label for="feedback">Feedback:</label>
    <textarea id="feedback" name="feedback"></textarea><br>

    <button type="submit">Submit</button>
</form>

3. Form with Checkboxes and Textarea

In forms, checkboxes can be used when users need to select multiple options. A <textarea> is useful for multi-line text input.

Example 3: A form for subscribing to a newsletter with multiple interests (checkboxes) and a message (textarea).

<form action="/subscribe" method="POST">
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required><br>

    <label>Interests:</label>
    <input type="checkbox" name="interest" value="tech"> Tech
    <input type="checkbox" name="interest" value="music"> Music
    <input type="checkbox" name="interest" value="sports"> Sports<br>

    <label for="message">Message:</label>
    <textarea id="message" name="message"></textarea><br>

    <button type="submit">Subscribe</button>
</form>
