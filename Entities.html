Using HTML Entities

HTML entities are used to represent characters that have special meaning in HTML or characters that are not available on the keyboard. Instead of using the actual character, you can use the corresponding entity name or numeric code.

1. Basic HTML Entities

Here are a few common HTML entities:

  • <: Less-than symbol
  • >: Greater-than symbol
  • &: Ampersand symbol
  • ": Double quotation mark

Example 1: Displaying special characters using HTML entities.

<p>This is an example of < and > symbols using HTML entities.</p>

Result:

This is an example of < and > symbols using HTML entities.

2. HTML Entities for Non-Printable Characters

There are many non-printable or special characters that can be used with HTML entities. For example, the copyright symbol or a trademark symbol:

  • ©: Copyright symbol (©)
  • ®: Registered trademark symbol (®)

Example 2: Displaying copyright and registered trademark symbols using HTML entities.

<p>The © symbol represents copyright.</p>
<p>The ® symbol represents a registered trademark.</p>

Result:

The © symbol represents copyright.

The ® symbol represents a registered trademark.

3. Using Numeric HTML Entities

Numeric entities allow you to represent characters using their Unicode values. These can be either decimal or hexadecimal:

  • ©: Copyright symbol using decimal code
  • ®: Registered trademark symbol using hexadecimal code

Example 3: Using numeric HTML entities to display characters.

<p>The © symbol represents copyright.</p>
<p>The ® symbol represents a registered trademark.</p>

Result:

The © symbol represents copyright.

The ® symbol represents a registered trademark.

4. HTML Entities for Mathematical Symbols

HTML entities can also be used to display mathematical symbols like plus-minus or multiplication signs:

  • ±: Plus-minus symbol (±)
  • ×: Multiplication sign (×)

Example 4: Displaying mathematical symbols using HTML entities.

<p>The result is ±5.</p>
<p>The value is 6 × 7 = 42.</p>

Result:

The result is ±5.

The value is 6 × 7 = 42.
