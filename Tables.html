Creating and Using Tables in HTML

Tables in HTML are used to display tabular data in rows and columns. Below, we'll explore how to create tables and use them effectively in HTML, with two examples for each.

1. Basic Table Structure

A basic table in HTML is created using the following tags:

  • <table> - Defines the table element.
  • <tr> - Defines a row in the table.
  • <th> - Defines a header cell in the table (bold and centered by default).
  • <td> - Defines a regular data cell in the table.

Example 1: A simple table with headers and data.

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
        <td>New York</td>
    </tr>
    <tr>
        <td>Alice</td>
        <td>30</td>
        <td>Los Angeles</td>
    </tr>
</table>

Result: A simple table with 2 rows of data.

Name Age City
John 25 New York
Alice 30 Los Angeles

2. Table with Columnspan and Rowspan

The colspan attribute is used to merge cells horizontally (across columns), while rowspan merges cells vertically (across rows).

Example 2: A table that spans columns and rows.

<table>
    <tr>
        <th colspan="2">Personal Information</th>
        <th rowspan="2">Location</th>
    </tr>
    <tr>
        <td>Name</td>
        <td>John Doe</td>
    </tr>
    <tr>
        <td>Age</td>
        <td>30</td>
        <td>New York</td>
    </tr>
</table>

Result: A table where the header spans two columns and the location cell spans two rows.

Personal Information Location
Name John Doe
Age 30 New York

3. Table with Caption

The <caption> tag is used to provide a title for the table. It is typically displayed above the table.

Example 3: A table with a caption.

<table>
    <caption>Student Information</caption>
    <tr>
        <th>Name</th>
        <th>Grade</th>
    </tr>
    <tr>
        <td>John</td>
        <td>A</td>
    </tr>
    <tr>
        <td>Jane</td>
        <td>B</td>
    </tr>
</table>

Result: A table with a caption and data rows.

Student Information
Name Grade
John A
Jane B

4. Table with Styles

Using CSS, we can style tables to make them look more presentable.

Example 4: A styled table with borders, padding, and background color.

<style>
    table {
        width: 100%;
        border-collapse: collapse;
    }
    th, td {
        padding: 10px;
        border: 1px solid #ddd;
        text-align: left;
    }
    th {
        background-color: #f4f4f4;
    }
</style>
<table>
    <tr>
        <th>Product</th>
        <th>Price</th>
    </tr>
    <tr>
        <td>Laptop</td>
        <td>$999</td>
    </tr>
    <tr>
        <td>Phone</td>
        <td>$599</td>
    </tr>
</table>

Result: A styled table with padding, borders, and background color.

Product Price
Laptop $999
Phone $599
