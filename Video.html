Using Video in HTML

The <video> element in HTML is used to embed video content on a web page. It allows for various attributes like controls for user interaction, autoplay to start the video automatically, and loop to make the video repeat. Below are some examples of how to use videos in HTML.

1. Basic Video with Controls

The <video> tag can include the controls attribute, which gives users the ability to play, pause, and adjust the volume of the video.

Example 1: A video with playback controls.

<video controls>
    <source src="video/sample.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

2. Video with Autoplay and Loop

The autoplay attribute makes the video play automatically when the page loads. The loop attribute makes the video loop once it reaches the end.

Example 2: A video that plays automatically and loops continuously.

<video autoplay loop>
    <source src="video/sample.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

3. Multiple Video Formats for Compatibility

For better browser compatibility, you can provide multiple formats for the video source, such as MP4, WebM, and Ogg.

Example 3: Providing multiple video formats for wider compatibility.

<video controls>
    <source src="video/sample.mp4" type="video/mp4">
    <source src="video/sample.webm" type="video/webm">
    <source src="video/sample.ogv" type="video/ogg">
    Your browser does not support the video tag.
</video>

4. Custom Video Controls Using JavaScript

With JavaScript, you can customize video controls, such as creating your own play/pause buttons, adjusting volume, and more.

Example 4: Custom play/pause button for video.

<video id="myVideo">
    <source src="video/sample.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

<button onclick="togglePlayPause()">Play/Pause</button>

<script>
function togglePlayPause() {
    var video = document.getElementById("myVideo");
    if (video.paused) {
        video.play();
    } else {
        video.pause();
    }
}
</script>
