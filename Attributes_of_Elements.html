Popular HTML Attributes

HTML elements can be customized using various attributes. Below are some of the most common HTML attributes, with examples:

1. "id" Attribute

The id attribute is used to uniquely identify an HTML element within a page. It can be used for styling with CSS or targeting with JavaScript.

Examples:

<div id="header">This is the header</div>
<p id="intro">This is an introductory paragraph.</p>

2. "class" Attribute

The class attribute is used to apply styles to multiple elements or group them together for JavaScript manipulation.

Examples:

<div class="box">This is a box.</div>
<p class="highlight">This paragraph is highlighted.</p>

3. "href" Attribute

The href attribute is used with the <a> tag to specify the URL of a link.

Examples:

<a href="https://www.example.com">Go to Example</a>
<a href="mailto:someone@example.com">Send an Email</a>

4. "src" Attribute

The src attribute is used with the <img>, <audio>, and <video> tags to specify the source of media content.

Examples:

<img src="image.jpg" alt="Description of the image">
<audio src="audio.mp3" controls></audio>

5. "alt" Attribute

The alt attribute provides alternative text for an image in case it cannot be displayed.

Examples:

<img src="image.jpg" alt="A beautiful sunrise">
<img src="logo.png" alt="Company Logo">

6. "style" Attribute

The style attribute is used to add inline CSS styles to an HTML element.

Examples:

<p style="color: blue; font-size: 20px">This text is blue and large.</p>
<div style="background-color: #f4f4f4; padding: 10px">This is a styled div.</div>

7. "target" Attribute

The target attribute is used with the <a> tag to specify how to open a link (e.g., in a new tab or the same tab).

Examples:

<a href="https://www.example.com" target="_blank">Open in a new tab</a>
<a href="https://www.example.com" target="_self">Open in the same tab</a>

8. "placeholder" Attribute

The placeholder attribute provides a short hint within a text field, describing the expected value or format.

Examples:

<input type="text" placeholder="Enter your name">
<input type="email" placeholder="Enter your email">

9. "value" Attribute

The value attribute specifies the value of an input element (e.g., a text field or button).

Examples:

<input type="text" value="John Doe">
<button value="Submit">Submit</button>

10. "width" and "height" Attributes

The width and height attributes are used to specify the dimensions of an element, commonly used with images, video, and canvas elements.

Examples:

<img src="image.jpg" width="300" height="200">
<video width="640" height="360" controls></video>
