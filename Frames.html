Introduction to Frames in HTML

Frames in HTML are used to divide the web page into different sections, each of which can display a different HTML document. However, frames are an outdated feature in HTML5. They have been replaced by other layout techniques like CSS Grid and Flexbox.

1. Using <frameset> (Deprecated)

The <frameset> tag is used to create a page with multiple frames. The <frameset> tag replaces the <body> tag in a frameset layout.

Example 1: Creating a webpage with two frames, one for navigation and one for content.

<html>
  <frameset cols="200px,*">
    <frame src="menu.html">
    <frame src="content.html">
  </frameset>
</html>

Result: The page will display two sections:

  • The left section will load menu.html (200px width).
  • The right section will load content.html and take up the remaining space.

Note: The <frameset> tag is deprecated in HTML5. It is recommended to use modern layout techniques like CSS Grid or Flexbox instead of frames.

2. Using <iframe> (Recommended)

The <iframe> tag is the modern way to embed another HTML document inside the current page. It is more flexible and widely supported compared to <frameset>.

Example 2: Embedding a YouTube video using an <iframe> tag.

<iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ" frameborder="0" allowfullscreen></iframe>

Result: This will display an embedded YouTube video on the page.

Example 3: Embedding a webpage inside your page using <iframe>.

<iframe src="https://www.example.com" width="600" height="400"></iframe>

Result: This will embed the website "example.com" inside the iframe with specified width and height.
