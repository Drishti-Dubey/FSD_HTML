Text-Related Elements in HTML

Text-related elements are fundamental to organizing and styling textual content in HTML. Below are some of the most common text-related elements, with two examples for each:

1. Headings

Headings are used to define the titles or subheadings in a webpage. HTML has six levels of headings, from <h1> to <h6>.

Examples:

<h1>This is a main heading (H1)</h1>
<h2>This is a subheading (H2)</h2>

2. Paragraph

The <p> tag is used to define a paragraph of text. It automatically adds space above and below the text.

Examples:

<p>This is a paragraph of text.</p>
<p>HTML stands for Hypertext Markup Language.</p>

3. Strong

The <strong> tag is used to indicate that a piece of text is of strong importance, which by default appears as bold.

Examples:

<strong>This text is bold because it is strong.</strong>
<p>Please read the <strong>important</strong> instructions carefully.</p>

4. Emphasis

The <em> tag is used to emphasize text, which by default appears as italicized text.

Examples:

<em>This text is emphasized and italicized.</em>
<p>The <em>quick brown fox</em> jumps over the lazy dog.</p>

5. Unordered List

The <ul> tag is used to create an unordered (bulleted) list, and the <li> tag defines each list item.

Examples:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

6. Ordered List

The <ol> tag is used to create an ordered (numbered) list, and the <li> tag defines each list item.

Examples:

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

7. Blockquote

The <blockquote> tag is used for a block of text that is a quotation from another source.

Examples:

<blockquote>
    "The only way to do great work is to love what you do." - Steve Jobs
</blockquote>

8. Code

The <code> tag is used to display inline code or commands.

Examples:

<p>Use the <code>alert()</code> function to display a message.</p>
<p>The command <code>git commit -m "Message"</code> is used to commit changes in Git.</p>

9. Horizontal Rule

The <hr> tag is used to create a horizontal line or rule, often used to separate sections of content.

Examples:

<hr>
<p>This is a section above the horizontal line.</p>
<hr>
<p>This is a section below the horizontal line.</p>

10. Line Break

The <br> tag is used to insert a line break, starting the text from the next line.

Examples:

<p>This is the first line.<br>This is the second line after a line break.</p>
<p>Name: John Doe<br>Email: john.doe@example.com</p>
