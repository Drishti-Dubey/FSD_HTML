Creating and Using Hyperlinks in HTML

Hyperlinks are used in HTML to link to other web pages, files, or resources. The <a> (anchor) tag is used to create hyperlinks. Below are examples showing different ways to create and use hyperlinks in HTML.

1. Linking to Another Web Page

The most common way to use hyperlinks is to link to other web pages. The <a> tag is used with the href attribute to define the destination URL of the link.

Example 1: A hyperlink to an external website (e.g., Google).

<a href="https://www.google.com">Go to Google</a>

Result:

Go to Google

Example 2: A hyperlink to a different page on the same website (e.g., About Us page).

<a href="/about-us">About Us</a>

Result:

About Us

2. Linking to an Anchor Within the Same Page

You can link to specific sections of the same page using anchors. An anchor link uses the # symbol followed by the id of the target element.

Example 1: A link to a specific section on the same page.

<a href="#section2">Go to Section 2</a>

Result:

Go to Section 2

Example 2: Another anchor link to a section further down on the page.

<a href="#section3">Go to Section 3</a>

Result:

Go to Section 3

Section 2
