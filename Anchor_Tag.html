Creating and Using Anchors in HTML

Anchors in HTML are used to link different parts of a page, or even to external web pages. The <a> tag (anchor tag) is used for creating hyperlinks that navigate to a specific location on the same page or to an external location.

1. Linking to a Specific Section Within the Same Page

We can create a hyperlink that points to a specific section within the same page using anchors. The link will jump to the location where the anchor is defined.

Example 1: A link that jumps to a section titled "About Us".

<a href="#about">Go to About Us Section</a>

Result:

Go to About Us Section

Here is the "About Us" section:

This is the About Us section. You can navigate here from the link above.

2. Using Anchors for Navigation to Other Pages

Anchors can also be used to link to other pages on the same website or to external websites. The href attribute specifies the destination URL.

Example 2: A link to an external website (Google) using an anchor.

<a href="https://www.google.com" target="_blank">Visit Google</a>

Result:

Visit Google

Example 3: A link to a contact page within the same website.

<a href="/contact">Go to Contact Page</a>

Result:

Go to Contact Page

3. Creating a Link to Scroll Back to the Top

We can also use anchors to create a "Back to Top" link that scrolls the page back to the top. This is useful for long web pages.

Example 4: A "Back to Top" link.

<a href="#top">Back to Top</a>
