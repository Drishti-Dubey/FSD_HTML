Using Audio in HTML

HTML provides the <audio> element to embed audio files. You can add various attributes such as controls for user interaction, autoplay for automatic play, and loop for looping the audio.

1. Basic Audio Playback with Controls

The <audio> element can be used with the controls attribute to allow the user to control the playback (play, pause, volume, etc.).

Example 1: Adding an audio file with playback controls.

<audio controls>
    <source src="audio/sample.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

2. Audio with Autoplay and Loop

The autoplay attribute allows the audio to start automatically when the page loads, and the loop attribute allows the audio to play repeatedly.

Example 2: An audio that plays automatically and loops continuously.

<audio autoplay loop>
    <source src="audio/sample.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

3. Multiple Audio Formats for Compatibility

For better compatibility across browsers, you can provide multiple audio sources with different formats. This ensures that the audio plays regardless of the user's browser support for certain formats.

Example 3: Providing both MP3 and Ogg formats for wider browser support.

<audio controls>
    <source src="audio/sample.mp3" type="audio/mpeg">
    <source src="audio/sample.ogg" type="audio/ogg">
    Your browser does not support the audio element.
</audio>

4. Customizing Audio Controls Using JavaScript

JavaScript can be used to control the audio, such as playing, pausing, changing volume, and more.

Example 4: Using JavaScript to play or pause audio.

<audio id="myAudio">
    <source src="audio/sample.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

<button onclick="toggleAudio()">Play/Pause Audio</button>

<script>
function toggleAudio() {
    var audio = document.getElementById('myAudio');
    if (audio.paused) {
        audio.play();
    } else {
        audio.pause();
    }
}
</script>
