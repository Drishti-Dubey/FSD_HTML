Using Images in HTML

In HTML, images are added using the <img> tag. The <img> tag is a self-closing tag, meaning it doesn't require a closing tag. To specify the image source, the src attribute is used. You can also specify the image's width, height, and alt text for accessibility.

1. Embedding an Image from a URL

Images can be added from external sources using the src attribute with a URL pointing to the image file.

Example 1: Embedding an image from an external URL (using a URL to an image on the web).

<img src="https://via.placeholder.com/600x300" alt="Placeholder Image">

2. Embedding an Image from a Local Directory

Images can also be linked from your local project folder or server. In this case, the src attribute contains a relative path to the image file.

Example 2: Embedding an image from a local directory (assuming the image is in the same directory as the HTML file).

<img src="local-image.jpg" alt="Local Image">

Result:

3. Resizing Images

You can control the size of an image using the width and height attributes, or by using CSS styles. This helps ensure that images fit within the layout of your page.

Example 3: Resize an image using the width and height attributes.

<img src="https://via.placeholder.com/600x300" alt="Placeholder Image" width="300" height="150">

Example 4: Resize an image using CSS for better responsiveness.

<img src="https://via.placeholder.com/600x300" alt="Placeholder Image" style="width: 50%; height: auto;">
