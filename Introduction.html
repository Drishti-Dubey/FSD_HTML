Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create and design webpages. It structures the content and adds meaning to it using tags. In this page, we will look at four basic examples of HTML tags:

1. HTML Heading Tags

The <h1> to <h6> tags are used for headings. <h1> is the largest, and <h6> is the smallest.

Example:

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
<h3>This is a Heading 3</h3>

2. HTML Paragraph Tag

The <p> tag is used to define paragraphs in HTML.

Example:

<p>This is a paragraph of text.</p>

3. HTML Anchor Tag

The <a> tag is used to create links. It is typically used with the href attribute to specify the URL.

Example:

<a href="https://www.example.com">Click here to visit Example.com</a>

4. HTML Image Tag

The <img> tag is used to embed images. It requires the src attribute to specify the image file source and alt for alternative text.

Example:

<img src="image.jpg" alt="Description of the image">
